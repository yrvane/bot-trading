# 🤖 Bot Trading - Stratégie VWAP

Bot de trading automatique pour S&P 500 et Or basé sur la stratégie VWAP.

## 📋 Prérequis

- Python 3.11+
- Compte démo MT5 (optionnel)
- API Finnhub (pour les news)

## 🚀 Installation

```bash
git clone https://github.com/yrvane/bot-trading.git
cd bot-trading
python -m venv trading_bot_env
source trading_bot_env/bin/activate  # ou trading_bot_env\Scripts\activate
pip install -r requirements.txt
